# Final Project
## Project Description: 
This is a single-player, terminal-based Number Guessing Game in Python. The program generates a random number based on the selected difficulty level, and the player attempts to guess it. After each guess, the program provides feedback indicating whether the guess is too high or too low. The game tracks the number of attempts and allows the user to replay. 

## How to Run the Program:
    - Download or clone the project files
    - Open a terminal
    - Navigate to where the file is saved
    - Run the program
    - Follow the prompts in the terminal
